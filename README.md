Interaction Methods

Hand Gestures
Use your index finger to draw
Press 'd' to start/stop drawing

Voice Commands
"start drawing"
"stop drawing"
"improve drawing"
"make 3D"
"show model"

Keyboard Shortcuts
'd': Toggle drawing
'i': Improve sketch
'm': Convert to 3D
's': Show 3D model
'c': Clear canvas
'q': Quit application
